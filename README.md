# JS Snacks - Funzioni

Raccolta di piccoli esercizi JavaScript ("snacks") con funzioni e manipolazione di stringhe.

## Contenuti
- 01_saluti
- 02_iniziali
- 03_filtra_iniziali
- 04_conta_vocali
- 05_bonus

Ogni cartella contiene un file `index.html` e uno `script.js` con la soluzione.

## Requisiti
- Browser moderno (Chrome, Firefox, Edge, Safari)

## Come eseguire
1. Apri la cartella dello snack che vuoi provare.
2. Apri `index.html` nel browser.
3. Controlla la Console del browser per l'output.

## Struttura del progetto
```
.
├── 01_saluti
├── 02_iniziali
├── 03_filtra_iniziali
├── 04_conta_vocali
└── 05_bonus
```

## Note
Gli esercizi sono pensati per praticare:
- funzioni
- array e stringhe
- cicli e condizioni
